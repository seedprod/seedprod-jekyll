The Press Kit below is for bloggers or anyone else wanting information on SeedProd's Coming Soon Pro plugin for WordPress.

Synopsis

SeedProd's Coming Soon Pro was created with goal of becoming WordPress's quintessential Coming Soon and Maintenance Mode plugin. The initial version launched in the WordPress.org repository back in July of 2011. Since that time the "Ultimate Coming Soon Page" and it's sister plugin "Ultimate Maintenance Mode" have risen to be the top plugins in their class and have amassed over 60,000 downloads. A pro version was created in late 2011 with a richer features set, more active roadmap and development timeline.

Coming Soon Pro now has a wide range options and integrations with more being added based on user feedback. Version 2 was just released and is now stable with an impressive feature set and Version 3 is already in works.
