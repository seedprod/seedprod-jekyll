No matter which WordPress Theme you use SeedProd's Coming Soon Pro will work with it.

Visible only to non logged in Users

Give your clients a secret link that allows them to bypass the Coming Soon page and view the website without logging in.

Search Engine Optimized

SeedProd's Coming Soon Pro HTML markup is optimized for search engines using the latest techniques. Plus you can customize the Meta Title and Meta Description to target your keywords.

Mobile Ready

Maintenance Mode automatically takes a screenshot of your website and overlays your downtime message. A proper 503 header status is returned so search engines know your site is unavailable.
