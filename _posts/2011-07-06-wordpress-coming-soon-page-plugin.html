Updated - Sept 14 2011 Pro Version now available with MailChimp Integration, Mobile Support and more.

I released a WordPress plugin a couple of weeks ago called the "Ultimate Coming Soon Page" plugin. My hopes with this plugin was to create an easy way for people to add a great looking coming soon page to their WordPress site as they built it out and to collect emails form their visitor. So far I've received excellent feedback from the user of it. I plan to release a pro version of the plugin in the next few weeks to be able to provide better features and support. If there's a feature you'd like to see in this plugin please leave feedback below.
